Podstawowe znaczniki HTML to:

  • <html> - Tu będzie umieszczony krótki opis
  • <head> - nagłówej strony z informacjami dla przeglądarek
  • <body> - właściwa treść dokumentu widoczna w oknie przeglądarki
  • <div> - element blokowy
  • <p> - akapit
  • <ol>,<ul> i <li> - element listy wypunktowanej lub numerowanej
  • <img> - obrazek
  • <a> - hiperłącze
